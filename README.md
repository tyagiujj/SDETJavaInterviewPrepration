# SDET Java Interview Preparation

## About This Repository

This repository contains Java programming concepts, coding practice, and interview preparation programs designed for SDET and QA Automation interviews.

The repository will be continuously updated with:
- Core Java concepts
- Java interview programs
- OOPs concepts
- Collections Framework
- Exception Handling
- String Programs
- Array Programs
- Logical Programs
- Coding Challenges
- Automation-related Java concepts

This project is created to strengthen Java fundamentals for Automation Testing and SDET roles.
